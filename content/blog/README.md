# Blog

This directory contains MDX files for blog posts migrated from Jekyll or newly created technical articles.

## File Structure

```
slug-name.mdx
```

## Frontmatter Template

```yaml
---
title: "Post Title"
slug: "post-slug"
description: "Brief description"
date: "2024-01-15"
tags: ["Tag1", "Tag2"]
published: true
---
```

Blog posts will be migrated from Jekyll in Phase 4.
